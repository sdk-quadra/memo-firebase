# Vue CLI memo app（firebase）
* Vue CLIのmemoアプリ
* 保存先はfirebaseです

## セットアップ
```
$ npm install
```

## firebase
* 自身のfirebase設定をしてください
```
$ npm install -g firebase-tools
```
* firebaseへのログイン
```
$ firebase login
```


## 起動
```
$ npm run serve
```
